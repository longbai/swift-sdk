## 安装

通过 CocoaPods

```ruby
pod "QiniuSwift", "~> 7.0"
```

## 运行环境

| Qiniu SDK 版本 | 最低 iOS版本 | 最低 OS X 版本 | 最低 tvOS 版本 | 最低 watchOS 版本 | Notes          |
